# CEC Livestream Scheduling

This project automates livestream scheduling for CEC services using the YouTube API and Google Calendar.

## Features
- Schedule livestreams for Cantonese services
- Integrates with Google OAuth and calendar APIs
- Automatically creates and manages stream metadata

## Requirements
- Node.js
- Google API credentials (not included in this repo)

## Setup
1. Clone the repo
2. Run `npm install`
3. Set up your credentials and tokens (not committed for security)
4. Run `node schedule-cantonese-stream.mjs`
